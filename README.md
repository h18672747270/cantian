# cantian 后台管理系统

#### 介绍

cantian 后台管理系统是一个支持单页面的 vue3 项目

#### 软件架构

## Vite + Vue3 + Typescript + Tailwindcss + pinia

#### 安装教程

- 安装依赖

```
npm i
```

- 开发模式

```
npm dev
```

- 打包

```
npm build
```

#### 目录结构

```
 |-- .env.dev  开发环境变量配置
 |-- .env.prod 生产环境变量配置
 |-- .gitignore git忽略改动配置
 |-- README.md
 |-- tailwind.config.js  tailwind配置
 |-- tsconfig.json  ts配置
 |-- vite.config.ts  vite项目配置
 |-- src
 |   |-- assets  静态文件
 |   |-- components  全局组件
 |   |-- views  页面
         |-- index.html 首页
 |   |-- utils  工具函数
 <!-- cicd -->
```
